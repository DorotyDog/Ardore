# Ardore

A user-driven transpiler. Build your own programming language using only Notepad.

## What is Ardore?

Ardore translates code written in your custom syntax into C++ (or any language you configure). All language rules are stored in editable `.txt` files. No parser code. No BNF. No dependencies.
